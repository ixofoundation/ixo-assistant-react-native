# ixo-assistant-react-native

The IXO assistant library for react native

## Installation

```sh
npm install ixo-assistant-react-native
```

## Usage

```js
import { multiply } from "ixo-assistant-react-native";

// ...

const result = await multiply(3, 7);
```
